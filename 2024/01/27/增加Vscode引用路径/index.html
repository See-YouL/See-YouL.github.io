增加Vscode引用路径

发表于 2024-01-27

问题说明

在嵌入式开发中需要经常用到库函数(SPL), Vscode需要配置引用路径才能对函数名或变量进行跳转

解决思路

复制一份库到主工作区的根目录下, 设置头文件引用路径到该库中即可使得Vscode实现对库中函数名和变量名的识别和跳转

Vscode工作区文件树

原工作区结构如下图

  • 工作区
  • 工程文件夹
  • Doc(略)
  • Libraries
  • CMSIS
    - startup
    - stm32f10x.h等内核头文件
  • STM32F10x_StdPeriph_Driver
    - inc(设外头文件)
    - src
  • Project(略)
  • User(略)

修改Vscode工作区结构

复制Libraries文件夹到工作区根目录下, 现工作区如下

  • 工作区
  • 工程文件夹
  • Doc(略)
  • Libraries
  • CMSIS
    - startup
    - stm32f10x.h等内核头文件
  • STM32F10x_StdPeriph_Driver
    - inc(设外头文件)
    - src
  • Project(略)
  • User(略)
  • Libraries
  • CMSIS
  • startup
  • stm32f10x.h等内核库文件
  • STM32F10x_StdPeriph_Driver
  • inc(外设头文件)
  • src

在Vscode中进行配置

在vscode中按Ctrl+Shift+P 输入configuration, 如图选择C/C++编程配置(json)

在"includePath"后面增加"./Libraries/CMSIS/“和”./Libraries/STM32F10x_StdPeriph_Driver/inc", 如下图所示
